/player
Using this command you can check info about any account (even if it's banned) including fame points, masteries, equipped skins.
/maps
This command allows you to check maps (events) that are active in the game now. The twist is that you can check the maps EVEN 1 MONTH IN THE FUTURE as well as map maker winner map.
/mastery
Ever wondered why you still play a brawler after reaching Gold III mastery? Well now you can see that you're still progressing your brawlers! We've custom made mastery tiers so you can farm as many mastery points as you like and see that you're progressing!
